# Spec-Kit Architect Previewer
This extension for [Spec-Kit](https://github.com/github/spec-kit) allows you to preview the architectural impact and risks of proposed changes before implementation.

## Installation
Run the following command in your Spec-Kit project:
```bash
specify preset add [https://github.com/UmmeHabiba1312/spec-kit-architect-preview](https://github.com/UmmeHabiba1312/spec-kit-architect-preview)
